AdXchange

Project information

  • Category: App, ML & Database

Developed MarketHub, a React Native app similar to the basic OLX platform, enabling users to buy and sell goods effortlessly.
Implemented user-friendly features allowing users to list items for sale by uploading photos, adding descriptions, and setting prices.
Incorporated a sell page facilitating direct communication between buyers and sellers via messaging for inquiries or purchases.
Created a robust backend using Node.js with Express.js, coupled with MongoDB for efficient data storage and management.
Integrated Firebase Cloud Storage for seamless storage and retrieval of product photos, ensuring a smooth user experience.
Implemented real-time notifications using Firebase Cloud Messaging (FCM) to promptly notify users of new messages or inquiries.
